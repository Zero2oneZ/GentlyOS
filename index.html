GentlyOS v0.1.0

Portable Alpine-based operating system - Take your computers back.

Network Boot (PXE/iPXE)

#!ipxe
kernel http://YOUR_IP:8080/boot/vmlinuz-lts alpine_repo=http://dl-cdn.alpinelinux.org/alpine/v3.21/main modloop=http://YOUR_IP:8080/boot/modloop-lts ip=dhcp
initrd http://YOUR_IP:8080/boot/initramfs-lts
boot

USB Boot

Download and write to USB:

curl -O http://YOUR_IP:8080/boot/vmlinuz-lts
curl -O http://YOUR_IP:8080/boot/initramfs-lts
# Use Ventoy or dd to create bootable USB

Boot Files

FileSizePurpose
vmlinuz-lts11 MBLinux Kernel
initramfs-lts24 MBInitial RAM disk
modloop-lts193 MBKernel modules

After Boot

# Login as root (no password)
# Run installer:
setup-alpine

# Or automated:
setup-alpine -f http://YOUR_IP:8080/answers
